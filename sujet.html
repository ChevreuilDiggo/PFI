Magic: The Gathering

Magic: The Gathering est un jeu de carte à jouer et à collectionner inventé par Richard Garfield en 1993. Le jeu contient plus de 50 000 cartes différentes avec lesquelles chaque joueur doivent créer leur propre deck. La taille du deck change selon le format. Il existe différent format, mais les plus populaires sont Commander multi et Duel Commander (cEDH) ainsi que le format standard. Dans les formats plus populaires, le deck consiste de 60 cartes pour le format standard ainsi que 99 cartes + 1 commander pour le format cEDH. Le jeu consiste de 5 couleurs: Bleu, Rouge, Vert, Blanc, Noir. Les cartes peuvent aussi avoir plusieurs couleurs en même temps. Il existe aussi des cartes incolores (les artefact). Un type de mana supplémentaire est le mana de type Phyrexian. Cette mana permet de payer 2 points de vie au lieu de payer la couleur (exemple sur une futur page). Il existe aussi différents types de cartes: Créatures, Planeswalker, Terrain, Enchantement, Éphémére (Instant), Rituel (sorcery) ainsi que plusieurs autre. De plus, afin de garder un semblant d'équilibre dans les matchs et tournoi, Wizards of the Coast, les propriétaires des droits du jeu, se permettent de banir ou restreindre des cartes selon le format. Par exemple, la carte Sol Ring, une carte estimé à être dans 84% des decks créé dans le monde, est légale dans le format commander, mais elle est banni/non légale dans les autres format parce qu'elle est trop puissante. Chaque carte dans Magic ont ce qu'on appelle un "converted mana cost". Celui-ci représente le nombre de mana nécéssaire pour la jouer. Il existe différentes zone dans le jeux de Magic: le cimetière, la libraire (le deck) ainsi que l'exile. En cEDH, il a la zone supplémentaire qui est la "Command Zone". Chaque créature ont aussi une Puissance et une Résistence (puissance/résistance sur la carte (voir la carte plus bas pour référence)). Lorsque la résistence de la carte tombe a 0 ou moins, la créature est considéré` morte est elle ira dans le cimetière, sauf certaines conditions.

Commander

Le format commander est un favori personnel. Ce format ce joue généralement à 3 ou 4, mais il existe un format duel. Le deck d'un joueur consiste de 99 cartes + 1 commander. Le commander est une carte spéciale, et des restrictions s'applique sur ce que peut être un commander. Le commander est une cartes créature ou planeswalker légendaire. Mon commander préféré serait probablement Goro-Goro and Satoru. Le commander est différent des autres cartes. Lorsqu'il quitte le "battlefield", le joueur a le choix entre l'envoyer dans la zone où il est supposé aller ou bien l'envoyer dans la command zone. Le problème de l'envoyer dans la command zone est que le commander devra subir la "Commander tax". Chaque fois que le commander retourne dans la command zone, le joueur devra payer 2 mana supplémentaires (la couleur n'est pas importante) pour le rejouer (par exemple, le commander qui coutait 4 mana total a jouer coutera maintenant 6 mana total).
